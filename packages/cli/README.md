# @veewo/claw

Publishable `claw` CLI for `.claw` project bootstrap, planning, search, truth ingestion, and completion refresh.

Install:

```bash
npm install -g @veewo/claw
```

Then run:

```bash
claw init
claw context
claw plan write --task my-task --title "My task" --goal "Define the first task"
```
